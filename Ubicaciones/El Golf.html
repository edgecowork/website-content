Ubicado entre Isidora Goyenechea y Av. El Bosque Norte, Edge Cowork El Golf tiene tres pisos con fachada hacia San Sebastián. Puedes llegar caminando desde el metro El Golf o metro Tobalaba, con una ubicación perfecta para la persona que le gusta estar en un distrito de negocios. Este es el primer local de Edge Cowork, con una decoración rústica e industrial que te permite estar en un entorno serio de trabajo pero a la vez relajado. El espacio está al lado de restaurantes, bancos, y matrices de empresa grandes. El cowork tiene tres sectores para coworkers. En el primer piso hay un coffee bar para los miembros y recepción para visitas. El segundo piso incluye una variedad de escritorios para tanto trabajar sentado como parado. Tenemos una gran mesa redonda para 8 a 10 personas que pueden conectarse a trabajar tranquilamente. Este piso incluye un lounge cómodo para reuniones informales, como también para trabajar en otro entorno. Este piso incluye Lockers para guardar tus cosas, baños y una cabina privada para esas reuniones 1 a 1 o si necesitas tomar una llamada importante. El tercer piso incluye una variedad de oficinas privadas y un espacio privado con Dedicated Desks para los que quieren un escritorio fijo. Este piso además tiene una sala de reuniones para 6 personas, kichennette y baños.
